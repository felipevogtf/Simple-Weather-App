## Instalación

Instalar dependencias

```
npm install
```

web deploy

```
ionic serve
```

android deploy

```
ionic capacitor run android
```

## Build

Se necesita [Android SDK](https://developer.android.com/studio/).

Construir una apk debug para android

```
ionic capacitor build android
```
